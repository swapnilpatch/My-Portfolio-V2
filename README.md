## Setup

This is a single-file HTML portfolio with local assets. No build step required.

### To run locally
Open `index.html` in any browser. For videos and embeds to work correctly, serve it over HTTP:
```bash
python3 -m http.server 8080
```
Then open `http://localhost:8080`

## Deploy to GitHub Pages

1. Push this repo to GitHub (must be **public**)
2. Go to **Settings → Pages**
3. Set source to **Deploy from branch → main → / (root)**
4. Save — site goes live at `https://yourusername.github.io/repo-name/`

> Replace `yourusername` and `repo-name` with your actual GitHub username and repository name.
